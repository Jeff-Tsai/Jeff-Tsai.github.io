Hexo操作步骤

发表于 2019-02-21 | 更新于 2019-02-22 | 分类于 Hexo

  • 初始化一个 hexo 项目

    hexo init <项目名>

  • 新建一个文档

    hexo new "文档名"

  • 执行页面

    hexo s

  • 安装next

  • 启用 next 主题

打开项目根目录下的 _config.yml
将 theme: landscape 改成 theme: next

  • 语言设置

在Hexo根目录下的_config.yml 文件修改
language: zh-CN

  • 添加分类标签

在项目根目录下输入
hexo new page categories

在Hexo/source下会新生成一个新的文件夹categories，在该文件夹下会有一个index.md文件。
index.md中添加type: “categories”

为某一篇文章添加分类，在Hexo/source/_post目录下的具体文章的categories中添加分类即可

  • 将本地文件部署到GitHub

hexo clean
hexo d

  • markdown语法

    单行代码引用：
    ·代码·

整段代码引用：
···
代码
···

首行缩进：
段首前添加全角的两个空格
